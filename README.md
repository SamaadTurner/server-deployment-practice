# server-deployment-practice  

# Math  

This does math functions but in Express!  

# Installation  

npm install  
Set your PORT environment with an .env file  

PORT=3001  
# Usage  
Send a request:  

method: GET  
route: /sum  
params:  
  value1{Integer}  
  value2{Integer}
